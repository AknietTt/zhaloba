# React Frontend

Минимальный React-проект на Vite.

## Как запустить

1. Установить зависимости:
   ```bash
   npm install
   ```
2. Запустить развитие:
   ```bash
   npm run dev
   ```
3. Открыть приложение в браузере по адресу `http://localhost:5173`.

## Команды

- `npm run dev` — запуск dev-сервера.
- `npm run build` — сборка приложения.
- `npm run preview` — предпросмотр сборки.
- `npm run lint` — запуск ESLint.
